# Airspace Conflict Impact on Global Aviation

## Project Overview
This project analyzes the impact of airspace conflicts on global aviation operations.

The dashboard highlights:
- Airline financial losses
- Passenger disruptions
- Flight cancellations
- Airport diversions
- Reroute delays

## Tools Used
